# react-devexpress
Простой проект для отладки компонент DevExtreme

Выполнить `npm run install-all` (два package.json), далее открыть папку в VS Code и запустить отладку (можно ставить брейкпоинты и в клиентском коде, и в серверном).  
Более простой вариант запуска: `npm start`, отлаживать можно в браузере.  
Имеется серверная часть, есть код для работы с CustomStore (данные для TagBox хранятся на сервере и отдаются по необходимости).  
При работе с TagBox на все ответы с сервера стоит задержка в 1 секунду (для эмитации работы с БД).  
Клиентская часть помимо компонент devexpress для react подключает компоненты react-bootstrap.  
Собирается клиент с помощью Vite. Чтобы не заморачиваться все написано на JS без TS.
